<h1><b>BMW / MINI Bimmerlink Log Visualizer</b></h1>

<h2><b>What is Bimmerlink?</b></h2>
<p><a href="https://www.bimmerlink.app/" target="_blank">https://www.bimmerlink.app/</a></p>

<p>Bimmerlink is an application that reads real-time sensor data and vehicle telemetry from BMW and MINI cars through the OBD port. 
It allows users to capture time-series logs directly from the vehicle's electronic control units.</p>

<p>Sensor Summary
<a href="https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/doc/bimmerlink-sensor-summary.pdf">
https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/doc/bimmerlink-sensor-summary.pdf</p>

<p>In this project, I aimed to visualize the log files generated by Bimmerlink (exported as .csv) and convert them into a structured PDF report with charts and insights.</p>

<h2 style="color:#EE0000;"><b>Disclaimer</b></h2>

<p style="color:#EE0000;">
This is a small utility application designed to visualize Bimmerlink time-series logs. As such, it may contain inaccuracies or errors.<br>

<h2><b>The application will generate two outputs:</b></h2>

<hr>

<h3><b>1. TXT Output — Sensor List</b></h3>

<p>A .txt file listing <b>all sensor names</b> found inside the CSV log.<br>
Example output:<br>
<a href="https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/test_data/2025-07-29_21-50-54_bimmerlink_web_sensorlist.txt">
https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/test_data/2025-07-29_21-50-54_bimmerlink_web_sensorlist.txt
</a>
</p>

<h3><b>2. PDF Output — Visual Report</b></h3>

<p>A .pdf file containing charts for each <b>non-constant sensor</b>, visualized and correlated with <b>engine speed</b>.<br>
This report helps you quickly understand sensor behavior, anomalies, and performance trends.<br>
Example output:<br>
<a href="https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/test_data/2025-07-29_21-50-54_bimmerlink_web_report.pdf">
https://github.com/armanasci/bimmerlink_log_analysis/blob/main/test_data/2025-07-29_21-50-54_bimmerlink_web_report.pdf
</a>
</p>
