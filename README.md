<h1><b>BMW / MINI Bimmerlink Log Visualizer</b></h1>

<h2><b>What is Bimmerlink?</b></h2>
<p><a href="https://www.bimmerlink.app/" target="_blank">https://www.bimmerlink.app/</a></p>

<p>Bimmerlink is an application that reads real-time sensor data and vehicle telemetry from BMW and MINI cars through the OBD port. 
It allows users to capture time-series logs directly from the vehicle's electronic control units.</p>

<p>Sensor Summary
<a href="https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/doc/bimmerlink-sensor-summary.pdf">
https://github.com/armanasci/bimmerlink_log_analysis_web/blob/main/doc/bimmerlink-sensor-summary.pdf</p>

<p>In this project, I aimed to visualize the log files generated by Bimmerlink (exported as .csv) and convert them into a structured PDF report with charts and insights.</p>
